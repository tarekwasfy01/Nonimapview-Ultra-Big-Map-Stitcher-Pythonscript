This is a must have for everyone using NoniMapView for stitching map tiles. Download maps in high resolution from google maps with NoniMapView. 
Afterwards stitch the tiles together with the python scipt. This allows you to create maps that are too big to stitch with other programs.

NoniMapView is a .jar, hopefully we can implement the working stitching script in the jar at some point to get the biggest possible maps with this simple crawler.
